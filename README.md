# DerbyDash
## Your new strategic, casual game that can be run on any computer!
##### The following is a: first-person,runner game, built in Python using Pygame.

 What you do:
- YOU the player are a jockey on a horse at a horse-racing track. 
- First you visit the bar, then you race! 
- The more/stronger effects you get from drinks higher your multiplier. You earn points the longer you survive on the race track and stay clear of the obstacles and security guards.
